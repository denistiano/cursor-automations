## Visual directions (pick A, B, or C)

### A — IDE Night (recommended)

- **Background:** `#090b0f` charcoal  
- **Surface:** `#12161d` panels  
- **Accent:** `#818cf8` indigo + `#38bdf8` cyan highlights  
- **Type:** Inter / system sans; monospace for code snippets  
- **Imagery:** Terminal screenshots, agent logs, repo trees — real work, not stock “AI brain”  
- **Matches:** Current HQ dashboard — fast to extend to landing

### B — Workshop Warm

- **Background:** warm off-white `#f8f6f3`  
- **Accent:** deep teal `#0d9488` + amber CTA `#f59e0b`  
- **Type:** Same as A for dev credibility  
- **Imagery:** Classroom + laptops, Plovdiv office when ready  
- **Best if:** In-person cohort is primary story

### C — Minimal Mono

- **Black/white + single accent** (electric green `#22c55e`)  
- **Type:** Bold grotesk headlines  
- **Best if:** Denis wants stark differentiation from SoftUni’s colorful edu brand

**Reply:** `brand day2: visual=A` (or B/C/custom Pinterest/URL)
